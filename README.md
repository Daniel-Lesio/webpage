###  Wolfpack -  webpack starter 
**includes : Sass postcss babel liveserver**

**instalation :** 

**Clone repository :** 

git clone https://github.com/Daniel-Lesio/webpage.git your-app-name

**instal dependencies :**

npm install

**start working**

npm run start

and open your browser at http: // localhost: 8080

**commands**

**run dev mode :** 

npm run dev

Open your browser : 

**build form production :**

npm run build
